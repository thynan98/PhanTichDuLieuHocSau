- Mô tả: Chương trình sử dụng Python và thư viện Pandas để phân tích và xử lý dữ liệu điểm xét tuyển đại học của học sinh được lưu trữ trong file dulieuxettuyendaihoc.csv. Dữ liệu bao gồm điểm trung bình các môn học của học sinh qua các năm và điểm thi đại học.
- Công nghệ sử dụng: sử dụng ngôn ngữ python kèm 2 thư viện hỗ trợ là Pandas giúp xử lý, phân tích dữ liệu dạng bảng và Numpy hỗ trợ tính toán số học.
- Cách hoạt động:
1. Tìm hiểu nghiệp vụ dữ liệu
2. Nhập liệu bằng công cụ từ file excel
3. Xác định dữ liệu định tính và định lượng
4. Hiệu chỉnh các thang đo phù hợp và kiểu giá trị dữ liệu cho từng biến số
5. Hiệu chỉnh dữ liệu và xử lý dữ liệu thiếu
6. Chuyển đổi (transformation) dữ liệu theo khoảng cho trước
7. Tạo biến số phụ thuộc theo biến độc lập
8. Tạo biến định tính phân loại
